Archivos planos

Archivos Planos

Un archivo plano, también conocido como archivo de texto plano, es un tipo de archivo informático que almacena datos de texto sin ningún formato o estructura específicos. 

                
               
Los archivos planos suelen contener datos en forma de texto sin 
                formato, como números, letras y símbolos, 
                separados por algún tipo de delimitador, 
                como comas, tabulaciones o espacios.

Archivos XML

Archivos XML

Un archivo XML (Extensible Markup Language) es un tipo de archivo informático que almacena información estructurada en formato de texto plano. El XML utiliza etiquetas para definir diferentes tipos de datos y sus relaciones, lo que permite crear documentos con una estructura jerárquica y organizada.

Archivos XML

< libro> < title>100 años de soledad< /title> < author>Gabriel Garcia Marquez< /author> < /libro>

Archivos json

Un archivo JSON (JavaScript Object Notation) es un tipo de archivo informático que almacena información estructurada en formato de texto plano. 


          
Estructura de archivo json

Estructura de archivo json

[ { "id": 1, "name": "Leanne Graham", "username": "Bret", "email": "Sincere@april.biz" } ]

Que es Parquet

Parquet es un formato de archivo columnar utilizado para el almacenamiento y procesamiento de grandes conjuntos de datos en un clúster de computadoras.

Ejemplo de Parquet

import pandas as pd import matplotlib.pyplot as plt # Leer el archivo Parquet df = pd.read_parquet('archivo.parquet') # Agrupar los datos por una columna y contar el número de ocurrencias counts = df.groupby('columna').size().reset_index(name='count') # Visualizar los datos en un gráfico de barras counts.plot(kind='bar',x='columna', y='count') plt.show()
